# Static Resources App

Flutter-приложение, демонстрирующее работу со статическими ресурсами: изображениями и пользовательскими шрифтами.

##  Задание «Техническое задание (ТЗ)

Создание проекта:Создайте новый проект Flutter с названием static_resources_app.

Работа с изображениями:Найдите 3 изображения (например, формата PNG или JPEG) для отображения в приложении:image1.pngimage2.pngimage3.pngПоместите их в папку assets/images/ в проекте.Настройте pubspec.yaml для подключения изображений.

Работа со шрифтами:Добавьте пользовательский шрифт в приложение. Например, скачайте шрифт (в формате TTF/OTF) и подключите его.Шрифт должен быть сохранён в папке assets/fonts/.Настройте pubspec.yaml для подключения шрифта.

Интерфейс приложения:Создайте экран приложения, который:Отображает три изображения (в ряд или по вертикали).Использует подключённый шрифт для заголовка и текста.Реализует тему приложения с кастомным шрифтом.

Тестирование и демонстрация:Запустите приложение на эмуляторе или устройстве, сделайте скриншоты.
